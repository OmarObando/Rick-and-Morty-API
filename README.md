# Rick-and-Morty-API
API that consume rick and morty API


To use, copy the repository with git clone
Write in console to install packages:

npm i

To run the api use:

npm run dev

To run test use :

npm test

The endpoint is 

http://localhost:3200/api/character/
http://localhost:3200/api/character?page=N
(Replace N with a number of page)

You need Basic Auth:
user: admin
password: admin
